# 위성사업
* 위성사업
> - 한반도 지역 및 주변국을 준 실시간으로 감시할 수 있는 중/대형 위성 탑재용 전자광학 탑재체와 고성능 영상 레이다(SAR) 탑재체 공급
> - 군 위성 통신망 내 망제어기 및 다양한 단말 공급 -> 육/해/공군 합동 작전 효과 극대화
> - 다목적 실용 위성 3A호 IR 센서, 차세대 중형위성 1,2호 EO 탑재체, 다목적 실용 위성 7호, 7A호 EO/IR 탑재체 개발 완료
